## 3. Logic Engines: The Roofing Math Engine

### Geometric Analysis (Vector Math)
The system analyzes 3D property data (Google Solar API) by evaluating vertices $(x, y, z)$.
- **Eave**: External Edge + Horizontal ($|z1 - z2| < 0.1$). Calculates Gutter/Drip edge.
- **Rake**: External Edge + Sloped ($|z1 - z2| \geq 0.1$).
- **Ridge**: Shared Edge + Horizontal + Convex.
- **Hip**: Shared Edge + Sloped + Convex.
- **Valley**: Shared Edge + Concave (Determined via Dot Product vs. Centroid).

### Precision Pricing Pipeline
The final retail price follows the **Transparency Model**:

$$TotalRetail = \frac{(Material + Labor + Overhead)}{(1 - 0.25)}$$

- **Material**: Dynamically pulled from **A-05 Line Item Catalog**.
- **Labor**: Calculated by Square count $\times$ Pitch Tier Rate.
- **Waste Factor**: 
    - **Global 10%** standard.
    - **Facet Logic**: < 5 Facets = 5%; 5-15 = 10%; 36+ = 22.5%.
- **Pitch Tiers**: Standard (3/12-6/12), Steep (7/12-9/12), Critical (10/12+).
